# **30 MCQs With Answers **

---

### **1. What is the main purpose of dimensional modeling?**

A. Increase storage efficiency
B. Organize data for fast and simple queries
C. Replace OLTP systems
D. Store unstructured data
**answer: B**

---

### **2. Which modeling technique is the most popular for modern data warehouses?**

A. Inmon’s methodology
B. One big table modeling
C. Data Vault
D. Kimball’s Dimensional Modeling
**answer: D**

---

### **3. Why is Kimball’s approach widely adopted?**

A. Requires no ETL
B. It is simple and optimized for analytics
C. It only works with unstructured data
D. It eliminates fact tables
**answer: B**

---

### **4. Which of the following is NOT a benefit of Kimball modeling?**

A. High performance
B. Complex joins
C. Tooling support
D. Scalability
**answer: B**

---

### **5. What does the star schema use to connect fact and dimension tables?**

A. Foreign keys
B. Joins on text columns
C. Triggers
D. Stored procedures
**answer: A**

---

### **6. A fact table typically stores:**

A. Only descriptive data
B. Historical unstructured data
C. Quantitative or measurable data
D. Only high-level summaries
**answer: C**

---

### **7. A dimension table typically stores:**

A. Composite primary keys
B. Descriptive attributes
C. Large numeric facts
D. Only transaction IDs
**answer: B**

---

### **8. What determines the level of detail in a fact table?**

A. Partitioning
B. Granularity
C. Surrogate keys
D. Denormalization
**answer: B**

---

### **9. Which approach did Kimball popularize?**

A. Top-down
B. Bottom-up
C. Middle-out
D. Hybrid
**answer: B**

---

### **10. In a star schema, dimension tables are usually:**

A. Highly normalized
B. In 3NF
C. Denormalized
D. Stored as JSON
**answer: C**

---

### **11. The composite primary key in a fact table is typically:**

A. Surrogate natural key
B. Combination of dimension keys
C. Sequence number
D. Not allowed
**answer: B**

---

### **12. What type of data is stored in a data lake?**

A. Cleaned structured data
B. Raw unstructured & semi-structured data
C. OLAP cubes
D. Fact tables only
**answer: B**

---

### **13. The data warehouse stores:**

A. Raw data without cleaning
B. Volatile operational data
C. Cleaned, non-volatile, historical data
D. Only dimension tables
**answer: C**

---

### **14. Data marts in Kimball are created using:**

A. Fully normalized schemas
B. The star schema
C. E-R diagrams
D. Hadoop clusters
**answer: B**

---

### **15. OLTP systems usually contain:**

A. Historical data
B. Cleaned data
C. Raw real-time operational data
D. Denormalized fact tables
**answer: C**

---

### **16. In a star schema, the fact table is usually:**

A. The smallest table
B. The most frequently joined table
C. Not used by analysts
D. Optional
**answer: B**

---

### **17. Dimension tables help analysts by providing:**

A. Detailed numeric facts
B. Attributes to filter, group, and label facts
C. Real-time processing
D. Normalized data integrity
**answer: B**

---

### **18. Which is the correct order in the star-schema flow?**

A. OLAP → Data Marts → ODS
B. Data Lakes → OLTP → OLAP
C. Cleaning → Transformation → Data Warehouse → Data Marts → OLAP
D. Data Marts → Data Warehouse → Data Lake
**answer: C**

---

### **19. What does high granularity mean?**

A. Very aggregated facts
B. Very detailed facts
C. No dimension keys
D. Only monthly summaries
**answer: B**

---

### **20. What does denormalization in star schema result in?**

A. Slow queries
B. Complex joins
C. Faster performance
D. More tables
**answer: C**

---

### **21. Kimball’s methodology begins by building:**

A. The enterprise warehouse first
B. Data marts first
C. OLAP cubes first
D. Data lakes first
**answer: B**

---

### **22. A dimension table key is usually:**

A. Composite
B. A single surrogate key
C. A foreign key to facts
D. A hash key
**answer: B**

---

### **23. Fact tables are generated by:**

A. Business events
B. Data dictionary processes
C. Metadata catalogs

### **24. Which layer is responsible for cleaning raw source data?**

### **25. What type of schema does Kimball promote?**
