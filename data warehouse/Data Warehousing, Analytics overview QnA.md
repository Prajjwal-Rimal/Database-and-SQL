## **Data Warehouse MCQ – Questions & Answers**

---

### **1. Who is considered the father of the data warehouse?**

**Options:**
A. Ralph Kimball
B. Bill Inmon
C. Edgar Codd
D. Jeff Bezos

**✅ Answer:** B. Bill Inmon

---

### **2. EDW stands for:**

**Options:**
A. Enterprise Data Workflow
B. Enterprise Data Warehouse
C. Enhanced Distributed Warehouse
D. External Data Warehouse

**✅ Answer:** B. Enterprise Data Warehouse

---

### **3. A data lake contains mainly:**

**Options:**
A. Cleaned, structured data
B. Processed and summarized data
C. Raw, unstructured or semi-structured data
D. Metadata about data

**✅ Answer:** C. Raw, unstructured or semi-structured data

---

### **4. A data warehouse is primarily used for:**

**Options:**
A. Real-time transaction processing
B. Machine control systems
C. Reporting, analytics, and decision-making
D. Storing logs only

**✅ Answer:** C. Reporting, analytics, and decision-making

---

### **5. A data mart is:**

**Options:**
A. A full data warehouse
B. A subset of a data warehouse focused on a specific business area
C. A type of cloud storage
D. Only raw data

**✅ Answer:** B. A subset of a data warehouse focused on a specific business area

---

### **6. Which of the following is NOT a characteristic of a good data warehouse?**

**Options:**
A. Consistent
B. Well integrated
C. Time stamped
D. Frequently overwritten

**✅ Answer:** D. Frequently overwritten

---

### **7. The process of extracting raw data, converting it to useful form, and loading it into a warehouse is called:**

**Options:**
A. ODBC
B. ETL
C. RTOS
D. CDC

**✅ Answer:** B. ETL

---

### **8. A data warehouse is a:**

**Options:**
A. Single source of truth
B. System for editing operational data
C. Raw data repository
D. Backup storage

**✅ Answer:** A. Single source of truth

---

### **9. Which system provides real-time, up-to-date operational data that can feed into a DW?**

**Options:**
A. Data Mart
B. Data Lake
C. Operational Data Store (ODS)
D. Metadata Layer

**✅ Answer:** C. Operational Data Store (ODS)

---

### **10. Data in a data warehouse is usually:**

**Options:**
A. Normalized
B. Non-normalized / denormalized
C. Stored in Excel
D. Purged daily

**✅ Answer:** B. Non-normalized / denormalized

---

### **11. Which of the following is subject-oriented?**

**Options:**
A. Application database
B. Operational system
C. Data warehouse
D. CPU cache

**✅ Answer:** C. Data warehouse

---

### **12. “Time-Variant” means:**

**Options:**
A. Data changes every second
B. Historical data is stored with timestamps
C. Data cannot be accessed
D. The system works only at night

**✅ Answer:** B. Historical data is stored with timestamps

---

### **13. “Non-volatile” means:**

**Options:**
A. Data changes frequently
B. Data is removed often
C. Once loaded, data is not changed or deleted
D. Data is always real-time

**✅ Answer:** C. Once loaded, data is not changed or deleted

---

### **14. Which is true about metadata?**

**Options:**
A. It is raw data
B. It explains data origin, structure, and meaning
C. It is used only in data lakes
D. It is a backup copy

**✅ Answer:** B. It explains data origin, structure, and meaning

---

### **15. Which is NOT part of metadata components?**

**Options:**
A. Transformation maps
B. Extraction and relationship history
C. Transaction logs for customers’ purchases
D. Data ownership

**✅ Answer:** C. Transaction logs for customers’ purchases

---

### **16. In transformation mapping, changing 'M' to 'Male' is called:**

**Options:**
A. Key change
B. Attribute conversion
C. Encoding/reference table conversion
D. Algorithmic change

**✅ Answer:** C. Encoding/reference table conversion

---

### **17. The layer responsible for cleaning and transforming data before loading:**

**Options:**
A. Operational database layer
B. Metadata layer
C. Information access layer
D. Presentation layer

**✅ Answer:** C. Information access layer

---

### **18. Which of these is an ETL step?**

**Options:**
A. Filtering dashboards
B. Extracting data
C. Creating marketing campaigns
D. Making AI models

**✅ Answer:** B. Extracting data

---

### **19. Data from different naming conventions and file types becoming standardized refers to:**

**Options:**
A. Time variance
B. Integration
C. Summarization
D. Volatility

**✅ Answer:** B. Integration

---

### **20. A data warehouse deployment model where infrastructure is physically on-site is called:**

**Options:**
A. Cloud
B. On-premise
C. Hybrid
D. Stream-based

**✅ Answer:** B. On-premise

---

### **21. A data warehouse stores:**

**Options:**
A. Only structured data
B. Only real-time data
C. Cleaned, historical, integrated data
D. Only logs

**✅ Answer:** C. Cleaned, historical, integrated data

---

### **22. Which of the following is used for day-to-day operations?**

**Options:**
A. Application-oriented system
B. Subject-oriented system
C. Data warehouse
D. Data mart

**✅ Answer:** A. Application-oriented system

---

### **23. A data warehouse mainly supports:**

**Options:**
A. OLTP
B. OLAP
C. Video streaming
D. Hardware drivers

**✅ Answer:** B. OLAP

---

### **24. Summarized data means:**

**Options:**
A. Raw data
B. Fully detailed transactional logs only
C. Aggregated data for reporting
D. Deleted data

**✅ Answer:** C. Aggregated data for reporting

---

### **25. Changing column formats like “float to decimal” is an example of:**

**Options:**
A. Physical characteristics conversion
B. Encoding conversion
C. Logic transformation
D. Key change

**✅ Answer:** A. Physical characteristics conversion

---

### **26. The output of ETL is stored in the:**

**Options:**
A. Data dictionary
B. ODS
C. Data warehouse
D. Browser cache

**✅ Answer:** C. Data warehouse

---

### **27. Data marts are usually organized by:**

**Options:**
A. Operating system
B. File size
C. Business domain (sales, finance, etc.)
D. Network topology

**✅ Answer:** C. Business domain (sales, finance, etc.)

---

### **28. Which statement is true about data lakes?**

**Options:**
A. They store only structured data
B. They are faster than data warehouses by definition
C. They store raw, uncleaned data
D. They always follow a strict schema

**✅ Answer:** C. They store raw, uncleaned data

---

### **29. “Data about the data” refers to:**

**Options:**
A. ODS
B. Metadata
C. Data mart
D. Data lake

**✅ Answer:** B. Metadata

---

### **30. A data warehouse becomes "time-stamped" because:**

**Options:**
A. Time is stored with facts, enabling historical analysis
B. It must refresh every second
C. It depends on network time protocol
D. Time is needed for login

**✅ Answer:** A. Time is stored with facts, enabling historical analysis
