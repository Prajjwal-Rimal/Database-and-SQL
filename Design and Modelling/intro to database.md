## Introduction
- database is widely used in almost all the industries be it social network, bank, government, businesses
- everyone uses it if the work with peoples data

<u><h3>Types of database</h3></u>

1. Hierarchical Databases: 
    - uses hierarchical tree to represent data
    - data organized in a parent -child structure
    - single root to which all the other data is linked
2. Network database:
    - extension of hierarchical database
    - use graph structure and many to many relationship
    - has more than one root
3. Object Oriented Database:
    - stores data as objects
    - useful when working with object oriented languages
4. Cloud Database:
    - hosted on cloud platforms
    - scale able and flexible
5. Distributed Database:
    - data is store across multiple servers
    - can be relational or nosql
6. NoSql Database:
    - designed for unstructured, semi-structures, or large scale data
    - has multiple types document oriented, key value stored, column oriented, graph database 
7. Relational Database:
    - most popular
    - has a tabular structure a two dimensional structure
    - relation maintained between tables
    - uses SQL

<u><h3>Relational database</h3></u>

1. stores data in tables
2. uses primary keys to uniquely identify rows
3. uses foreign keys to relate tables to each other
4. data is accessed and managed using SQL
5. ensures data integrity with constraints (e.g., NOT NULL, UNIQUE)
6. supports ACID (atomicity, consistency, isolation, durability) properties
7. MySQL, PostgreSQL, Oracle, SQL Server
